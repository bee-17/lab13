# 🎬 CineSpoilerS

E-commerce de tickets de cine construido con React + Vite + TypeScript.

## 🛠️ Stack Tecnológico

- ⚛️ React 19 + Vite
- 🟦 TypeScript
- 🎨 Tailwind CSS v4
- 🧩 shadcn/ui
- 📦 Axios

## 🚀 Instalación

```bash
git clone https://github.com/bee-17/lab13.git
cd cinespoilers
npm install
npm run dev
```
